What is this?

discord-plus is a theme for Discord. It's both sleek and customizable, without getting in your way. Its top priority is balancing between looks and performance.

All you need to install my theme is Discord and a compatible client mod.

All you need to install my theme is a computer with Discord and a compatible client mod.

User customization

Several variables in the theme file can be customized:
  • Backdrop
  • Color
  • Font
  • Roundness
  • Padding
